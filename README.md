# React JS Interface

This is the main JavaScript React prototype for the project.

## Run

1. `npm install`
2. `npm run dev`

## Build

- `npm run build`

## TypeScript Backup

The original TypeScript prototype that was created with Figma has been moved to `typescript-backup`.

To run it:

1. `cd typescript-backup`
2. `npm install`
3. `npm run dev`
