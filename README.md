# React JS Interface (Simplified)

This is the main JavaScript React app for this repository.

## Run

1. `npm install`
2. `npm run dev`

## Build

- `npm run build`

## TypeScript Backup

The original TypeScript app has been moved to `typescript-backup`.

To run it:

1. `cd typescript-backup`
2. `npm install`
3. `npm run dev`
